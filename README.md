# Volo Yatra — Business Suite v3

Travel agency management system built for Replit + PostgreSQL.

## Setup

1. Fork/import this repo into Replit
2. Add `DATABASE_URL` and `JWT_SECRET` in Replit Secrets
3. Click Run — database initializes automatically

## Features
- Air / Hotel / Train / Visa / Tour / Taxi bookings
- GST invoicing with auto journal entry
- Multi-line journal entries (debit must equal credit)
- Bank statement PDF upload + ledger posting
- Portal wallets (MakeMyTrip, Booking.com, Razorpay)
- Party ledger download (PDF + Excel, full + pending only)
- CRM pipeline, Marketing campaigns, Analytics
- User management with edit + password change
